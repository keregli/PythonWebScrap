Project was developed and compilied using PyCharm.

Python version/Interpreter: Python 3.9

Once Project is extracted from zip folder, and opened in Pycharm, install the following packages:
requests
requests-html
pandas
seleium
BeautifulSoup

After installing packages, configure PyCharm to run \Project1\venv\wScraper1.py

Running the file should export all the data on all the pages into the requested csv file thread.csv

Directory for csv file should be \Project1\venv\thread.csv
